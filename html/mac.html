Терминал MacOS

  • ls - list
    ls -a — выводит все файлы в папке.
  • cd - change directory 
    cd ~ —  переход в домашнюю директорию.
  • pwd - present working directory 
    pwd — отображение текущей директории.
  • mkdir - make directory 
    mkdir — создает папку.
  • mv - move 
    mv — перемещает (переименовывает) файл.
